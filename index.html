    Curso práctico de JavaScript

    De la escuela JavaScript de Platzi

    Instruido por Juan David Castro

    Lo que veremos en este curso:

    Crear un repositorio en Git para guardar el registro histórico con todos los cambios de nuestro código.

    Utilizar funciones en JavaScript para encapsular el código, organizarlo y reutilizarlo.

    Realizar operaciones matemáticas directo en el código JavaScript.

    Utilizar la propiedad document.getElementById("") para rescatar información desde el formulario html.

    Crear ciclos if-else que evalúen diferentes condiciones.

    Diferenciar entre parámetros y argumentos, entendiendo que las funciones reciben los parámetros y los argumentos son enviados a las funciones.

    Aprender a utilizar diferentes métodos de JavaScript:

    • .map sirve para crear un nuevo arreglo a partir de iterar por cada elemento del arreglo inicial.
    • .reduce sirve para iterar por cada elemento de un arreglo y dar como resultado algún valor final.
    • .splice sirve para cortar alguna cantidad de elementos dentro de un arreglo.
    • .push sirve para insertar un nuevo elemento al final de un arreglo.
    • .length sirve para leer la cantidad de elementos de un arreglo.

    Y como un extra del curso, aprender a comentar un bloque de código completo. Para esto se debe seleccionar el bloque y luego presionar las teclas CTRL + K (y sin soltar la tecla CTRL) + C.
